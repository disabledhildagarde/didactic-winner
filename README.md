# My App

A Dynamics 365 Business Central extension written in AL (Application Language).

## Prerequisites

- Visual Studio Code
- AL Language extension for VS Code
- A Dynamics 365 Business Central instance (local or cloud)

## Getting Started

1. Clone this repository
2. Open the folder in VS Code
3. Update `app.json` with your extension details
4. Configure the launch settings in `.vscode/launch.json` to match your environment
5. Press `F5` to build and deploy

## Project Structure

- `app.json` - Extension manifest and configuration
- `src/` - AL source code
- `.vscode/` - VS Code workspace settings and launch configuration
